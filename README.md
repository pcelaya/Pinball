Gameplay:

- You have 3 lives, move the flippers to get the ball going.

Points:

 - Arms: 25p

 - Buu: 200p

 - Fists: 15p

 - Shenlong: 75p 

Controls:
 
 - RIGHT & LEFT ARROW: move the flippers
 
 - DOWN ARROW: throw the ball
 
 
 - Others:

   R: spawn extra balls

   LEFT CLICK: move the ball

   F1: show/hide colliders

   F2: change the target FPS value

   F3: less gravity

   F4: normal gravity

   F5: more gravity
